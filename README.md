# Chain Vault Frontend

This is the frontend application for **Chain Vault**, a decentralized or secure storage application. It is built with **React** and **Vite** to provide a fast, responsive, and modern user interface.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* **Node.js** (v16.0.0 or higher recommended)
* **npm** or **yarn**

### Installation

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/SithilSemitha/chain-vault-frontend.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd chain-vault-frontend
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

## 💻 Usage

### Development Server
Run the application in development mode with hot-reloading:
```bash
npm run dev
# or
yarn dev
```
The app will typically be available at `http://localhost:5173`.

### Production Build
To create an optimized production build:
```bash
npm run build
# or
yarn build
```

### Preview
To preview the production build locally:
```bash
npm run preview
```

* `src/` - Contains the main logic, components, and assets.
* `public/` - Static assets like icons and images.
* `index.html` - The entry point for the application.
* `vite.config.js` - Configuration for the Vite bundler.
